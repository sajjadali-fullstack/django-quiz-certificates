🔹 django-allauth

This is a third-party authentication package for Django called django-allauth.

It adds features like:

1. User registration

2. Login / logout

3. Email verification

4. Social login (Google, Facebook, GitHub, etc.)


=========================================================

🔐 What is a JWT?

A JWT (JSON Web Token) is a secure way to send information between two parties.

It is commonly used for:

Authentication

API security

Social logins (like Google login)

🧩 Why Your Django Project Needs It

You are using django-allauth with the Google provider.

When someone logs in with Google:

1. Google sends back a token.

2. Your app must verify that token.

3. That verification uses JWT.

4. JWT verification requires PyJWT.

That’s why Django crashes if PyJWT is missing.


📦 Simple Explanation

1. PyJWT = Python library

2. jwt = module name inside that library

3. It helps verify login tokens from providers like Google.

What is cryptography?

>>> cryptography is a Python security library.

It provides encryption tools that help applications securely:

. Encrypt and decrypt data

. Verify digital signatures

. Work with secure tokens (like JWT)

. Handle SSL / HTTPS security


🧠 Why Your Django Project Needs It

You’re using django-allauth with Google login.

When someone logs in with Google:

1. Google sends back a secure token.

2. Your app must verify that token.

3. Verifying it requires cryptographic functions.

4. Those functions come from the cryptography library.

So if it's missing → Django crashes.



🔐 Simple Real-World Example

Think of cryptography like a security lock toolkit:

. It checks if a “digital ID card” (Google login token) is real.

. It ensures no one tampered with it.

. It confirms it was actually issued by Google.

Without it, your app can’t safely trust login tokens.

📌 What is WeasyPrint?

WeasyPrint is a Python library used to:

. Convert HTML → PDF

. Generate reports

. Create downloadable PDF files from templates

It’s commonly used in Django projects to generate PDF documents (like invoices, quiz results, certificates, etc.).
